> Shorten URLs, track clicks globally, and gain insights with detailed analytics dashboards.

<table>
<tr>
<td width="50%">

**What it does**
- Instant URL shortening with custom aliases
- Real-time click analytics (geo, device, browser, referrer)
- Full auth system with password reset via email
- Rate limiting on all API endpoints
- SEO: dynamic OG images, JSON-LD, sitemap

</td>
<td width="50%">

**Engineering highlights**
- Edge Middleware for instant 301 redirects
- Fire-and-forget click tracking (zero redirect delay)
- Denormalized counters avoid expensive COUNT queries
- SHA-256 IP hashing for visitor privacy
- MongoDB aggregation pipelines for O(log n) analytics
- In-memory rate limiter with Retry-After headers
